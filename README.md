# Smart Food Donation System

A MERN Stack web application that helps connect food donors with NGOs to reduce food waste and support communities in need.

---

## About the Project

Every day, restaurants, hotels, caterers, event organizers, and individuals have surplus food that often goes to waste.

At the same time, NGOs and charitable organizations struggle to provide enough meals for people in need.

This project aims to provide a centralized platform where verified food donors and NGOs can connect quickly and safely.

---

## Project Objectives

- Reduce food wastage
- Connect food donors with NGOs
- Enable quick food donation requests
- Support verified NGO registration
- Build a simple and user-friendly donation platform

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
